Obsidian

发表于2022-04-27|更新于2024-01-18
|浏览量:

参考

Johnny学的个人空间-Johnny学个人主页-哔哩哔哩视频 (bilibili.com)

简介

  • 免费使用
  • 支持 Markdown 语法的知识库
  • 文件保存在本地，文件格式为 Markdown 文档，可迁移
  • 支持双相链接
  • 配合插件，可以与其他软件一同使用
  • 可以嵌入 Canvas 的图片内容
  • 可以嵌入 Bilibili 的视频内容
  • 可以嵌入本地的视频内容
  • 可以调用 Pandoc
  • 强大的搜索内容功能

插件学习

不要老是想着 all in Obsidian

  • Mind Map
    • 可以生成思维导图，便于查看大纲
  • Pandoc Plugin
    • 调用了 Pandoc，用来生成 PPT 等格式的文件
  • Note Refactor
    • 将大文档根据标题切分成小文档，并在大小文档之间建立链接
  • Spaced Repetition
    • 类似于 Anki，可以构建学习卡片
  • dataview
    • 把 Obsidian 变成数据库使用
  • Footnote shortcut
    • 可以使用快捷键在文段中加入角标注释，
  • Tiny Footnotes
    • 可以将文段中乱序的角标重新排序
  • Advanced URI
    • 用 uid 代替文件路径生成稳定的外链
  • Citations
    • 通过 BibLaTeX 的 bib 文件生成引文
  • QuickAdd
  • Excalidraw

其他

  • 幕布
    • 可以 zoom in 某一个条目之内进行编辑
  • workflowy
  • Airtable
    • OB社区插件汇总 每周更新
  • 飞书
    • 完全支持 Markdown 语法
  • Zotero
    • 文献管理，论文阅读
    • 可以导出笔记

无纸化学习
优点 好记性不如烂笔头。 平板上做的电子笔记，修改起来比纸质的笔记更容易。 软件选择 GoodNotes 可以嵌套文件夹，好 把 PPT/PDF 导入后，方便写写画画 把电子书上重点内容截图后，粘贴到新建稿纸上再继续写点东西 MarginNote3 可以把高亮的内容作为目录，把一本书组织起来 后记 记笔记非常重要，可惜我没能早点学会。在使用 iPad 做笔记之后，我逐渐明白了做笔记的意义。 很多细节只有老师上课讲的时候才会提到，比如推导的过程，或者说这里为什么要这么做，是怎么样想到的。这些细节需要立刻记录下来。但是以前的学习生活中我可以直接记住，直到记不住了才意识到笔记的重要性。 在这个意义上，纸质笔记和电子笔记的区别不大。电子笔记用多了之后，发现自己并不会经常去复习。所谓的方便查阅之类的有点都是一些”假“优点。
