Obsidian

发表于2022-04-27|更新于2024-01-18
|阅读量:

参考

Johnny学的个人空间-Johnny学个人主页-哔哩哔哩视频 (bilibili.com)

简介

  • 免费使用
  • 支持 Markdown 语法的知识库
  • 文件保存在本地，文件格式为 Markdown 文档，可迁移
  • 支持双相链接
  • 配合插件，可以与其他软件一同使用
  • 可以嵌入 Canvas 的图片内容
  • 可以嵌入 Bilibili 的视频内容
  • 可以嵌入本地的视频内容
  • 可以调用 Pandoc
  • 强大的搜索内容功能

插件学习

不要老是想着 all in Obsidian

  • Mind Map
    • 可以生成思维导图，便于查看大纲
  • Pandoc Plugin
    • 调用了 Pandoc，用来生成 PPT 等格式的文件
  • Note Refactor
    • 将大文档根据标题切分成小文档，并在大小文档之间建立链接
  • Spaced Repetition
    • 类似于 Anki，可以构建学习卡片
  • dataview
    • 把 Obsidian 变成数据库使用
  • Footnote shortcut
    • 可以使用快捷键在文段中加入角标注释，
  • Tiny Footnotes
    • 可以将文段中乱序的角标重新排序
  • Advanced URI
    • 用 uid 代替文件路径生成稳定的外链
  • Citations
    • 通过 BibLaTeX 的 bib 文件生成引文
  • QuickAdd
  • Excalidraw

其他

  • 幕布
    • 可以 zoom in 某一个条目之内进行编辑
  • workflowy
  • Airtable
    • OB社区插件汇总 每周更新
  • 飞书
    • 完全支持 Markdown 语法
  • Zotero
    • 文献管理，论文阅读
    • 可以导出笔记
