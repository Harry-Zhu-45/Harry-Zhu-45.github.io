Zotero 简介

  • 不仅是科研相关的论文管理软件，而是所有人都可以用的管理型软件
  • 文件管理视同过一条条数据实现的，操作就是写数据
  • 导出文献库

插件推荐

  • Zotfile
    • 参考：zotero文献管理器插件：ZotFile - 知乎 (zhihu.com)
  • Mdnotes
    • 参考：Zotero Mdnotes插件模板教程 - 知乎 (zhihu.com)
  • Better-BibTeX
    • 参考：Zotero | 06-如何在LaTeX写作中使用Zotero（Better-BibTeX的使用） 哔哩哔哩_bilibili
  • PDF-translate
    • 参考：解决问题：Zotero翻译神器推荐，超级好用 - 知乎 (zhihu.com)

可能的使用流程 (已弃用)

  • 用 Zotero 自带的 pdf 阅读器读文献，加高亮和注释
  • 右击文章的父目录 -> Mdnotes -> Create Mdnotes file
  • 右击 文章的父目录 -> 从注释中添加笔记
  • 右击 注释笔记 -> Export Note... -> Include Zotero Links
  • 最后把注释笔记剪切到文章的 Mdnotes file 中，完成！
